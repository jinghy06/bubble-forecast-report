## 📊 核心发现

通过将历史数据扩展至**1990年**，DTW滑动窗口分析揭示了一个**完全不同于初步分析**的结论：

**当前市场更像 1990-1997年的"泡沫酝酿期"（技术革命稳步渗透阶段），而非1999-2000年的"泡沫顶峰"（疯狂投机阶段）。**

| 指标 | 初步分析（历史始于1995） | 扩展分析（历史始于1990） |
|------|------------------------|------------------------|
| 最相似窗口 | 1995-2002（泡沫完整周期） | **1990-1997（泡沫酝酿期）** |
| 纳斯达克相似度 | Composite=0.401 | Composite=0.549 |
| 标普500相似度 | Composite=0.409 | **Composite=0.685** |
| 阶段判断 | 泡沫后期/顶峰附近 | **泡沫酝酿期早期** |
| 距泡沫顶峰 | 可能 imminent | 可能还有2-4年 |

## 📈 分析方法

### 1. 数据来源

- **Yahoo Finance**（通过 kimi_datasource_call_v2 API）
- 分批下载2年区间的历史数据，合并为完整时间序列
- 覆盖1990-2005年（纳斯达克缺失1994-1995数据，使用线性插值填补）

### 2. 数据处理

- **日线 → 周线**：取每周最后一个交易日的收盘价
- **归一化**：起点=100的百分比变化法

### 3. DTW滑动窗口分析

- **模板**：2018-2025年（约392周）
- **历史数据**：1990-2005年（纳斯达克约731周，标普500约835周）
- **滑动步长**：4周
- **综合相似度**：
  - `Composite = 0.45 × Pearson + 0.35 × DTW_sim + 0.20 × pct_change_corr`

### 4. DTW算法

- 自定义实现动态时间规整
- 使用 Sakoe-Chiba Band（`max(n,m)//4`）加速
- DTW距离通过指数衰减转换为相似度分数

## 📉 走势形态对比

| 维度 | 1990-1997（最相似） | 1995-2002（泡沫期） | 2018-2025（当前） |
|------|-------------------|-------------------|-----------------|
| 起步后走势 | 平缓稳步上涨 | 前半段上涨，后半段抛物线暴涨 | 平缓稳步上涨 |
| 中途回调幅度 | 10-20% | 20-30% | 15-25%（2020疫情、2022加息）|
| 末期形态 | 温和加速 | 疯狂加速后暴跌 | 温和加速 |
| 终点值（起点=100）| ~275-350 | ~130-200（含暴跌）| ~275 |
| 波动特征 | 低波动、趋势稳定 | 高波动、极端行情 | 中等波动、趋势较稳 |

**核心发现**：当前走势的"平缓稳步上涨+中途中等回调+末期温和加速"形态，与1990-1997年高度吻合，而与1995-2002年的"抛物线暴涨+暴跌"形态显著不同。
